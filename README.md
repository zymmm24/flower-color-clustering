# 花卉图像动态颜色聚类识别方案

## 一、项目背景

本项目面向花卉图像的分类识别场景，当前阶段主要以花卉的颜色特征作为分类依据，不将形状、纹理、花型结构等信息作为主要判定条件。由于花卉颜色本身具有自然渐变、相近色过渡明显、拍摄环境光照变化较大以及后续数据持续增加等特点，传统固定类别的分类模型难以适应实际业务需求。

在实际应用过程中，不同花卉之间可能存在颜色接近但肉眼难以区分的情况，如果仅依赖纯数值差异进行划分，会导致类别被过度拆分，降低分类稳定性。因此，本项目采用符合人眼视觉感知的颜色差异标准，将“人眼是否能够明显区分颜色差异”作为类别划分依据，以提高聚类结果的实际可用性和业务合理性。

项目整体采用动态聚类机制，在初始阶段自动发现颜色类别，在后续运行过程中持续接纳新增数据，并允许未知类别存在，从而保证系统能够随着数据规模增长持续扩展和自适应更新。

## 二、项目目标

- **自动初始化分类**：对现有花卉图像数据进行自动聚类，在类别数量未知的情况下自动形成初始颜色类别。
- **在线流式流转**：对后续持续流入的花卉图像进行在线分类，将其自动归入最接近的已有类别原型。
- **异常与未知防御**：建立未知类缓冲机制，当新图像与现有类别差异较大或处于模糊地带时，不强制归入已有类别，而是保留为未知类别候选。
- **自适应迭代更新**：定期对未知类别进行局部重聚类，动态发现新的颜色类别并更新系统类别结构，避免老类别标签产生漂移。
- **视觉感知拟合**：采用符合人眼视觉感知的颜色差异标准与度量空间，提高聚类稳定性与业务可解释性。

## 三、技术路线

本项目采用“多主色提取 + 统一色彩度量空间（EMD+ΔE00） + 距离矩阵聚类 + 增量式未知缓冲”的技术路线。

系统首先将花卉图像转换为 **CIELAB 色彩空间**，并使用修正后的 **CIEDE2000（ΔE00）** 公式作为底层色差代价值，通过调高明度惩罚项权重，降低因拍摄光照、阴影变化带来的干扰。

在特征表达上，系统放弃高维直方图拼接，采用 **MiniBatchKMeans** 提取图像的 $K$ 个核心主色及其面积权重，形成稀疏、高解释性的多主色特征组合。

为了保证离线聚类与在线识别在数学逻辑上的一致性，系统引入**陆地移动距离（EMD, Earth Mover's Distance）**，以 ΔE00 为基础代价函数，计算图像间的多主色分布差异。离线阶段直接向 **HDBSCAN** 输入预计算的 EMD 距离矩阵进行不定量聚类；在线阶段则基于 EMD 距离进行原型匹配与灰区拦截，动态维护未知类缓冲池并实施局部增量重聚类。

## 四、核心业务流程



**1.色彩空间转换与主色提取:**数据预处理阶段.

将输入的图像统一转换至 CIELAB 色彩空间。利用 MiniBatchKMeans 算法提取图像中核心的 3 到 5 个主色，并记录各主色在图像中的面积占比，构建形如 `[(Lab1, 权重1), (Lab2, 权重2), ...]` 的稀疏特征表达。

**2.构建非线性视觉距离矩阵:**统一通用度量.

以调整明度权重（令 kL = 2.0）后的 CIEDE2000（ΔE00）作为底层色差代价值。利用陆地移动距离（EMD）算法，计算两两样本间多主色分布的最小运输成本，生成全局样本的非线性视觉距离矩阵。

**3.离线距离矩阵聚类:**初始化类别原型.

将预计算的 EMD 距离矩阵直接输入 HDBSCAN 聚类算法。在不预设类别数量的前提下自动识别出离群噪声并形成密集的颜色簇。对每个聚类簇，计算其多主色质心及类内最大 EMD 距离，生成该类的“颜色原型”。

**4.在线流式分类与灰区拦截:**生产流转阶段.

新流入的图像按相同方式提取多主色。计算新样本与各已有类别原型的 EMD 距离：

- 距离小于等于同类阈值：直接归入该类，并采用滑动平均动态微调该类的原型中心。
- 距离处于灰区范围：拦截并送入未知缓冲池。
- 距离大于异类阈值：判定为新颜色候选，送入未知缓冲池。

**5.未知池局部增量重聚类:**动态自适应迭代.

当未知缓冲池中的样本数量或占比达到设定阈值时，触发局部重聚类。仅对缓冲池内的样本重新计算 EMD 距离矩阵并运行 HDBSCAN。若能形成稳定新簇且数量达标，则在主系统中新增颜色类别标签，存量老类别的结构和标签保持不动。



## 五、参数设置

### 5.1 色彩空间与色差感知标准

| **参数项**       | **设置值**                    | **业务含义与配置说明**                                       |
| ---------------- | ----------------------------- | ------------------------------------------------------------ |
| **色彩空间**     | CIELAB                        | 提供符合人类视觉感知均匀性的底层空间。                       |
| **色差计算公式** | CIEDE2000（ΔE00）             | 用于计算单点感知色差。通过设置权重参数 $k_L = 2.0, k_C = 1.0, k_H = 1.0$，有意降低系统对光照、高光及阴影明暗变化的敏感度。 |
| **同类判定阈值** | $\Delta E_{00} \le 2.5$       | 容忍正常拍照设备白平衡漂移与轻微阴影，此范围内的色差视为同类颜色。 |
| **灰区缓冲范围** | $2.5 < \Delta E_{00} \le 6.0$ | 视觉上存在相近或过渡趋势，但不确定性较高，需要拦截进入缓冲池观察。 |
| **异类判定阈值** | $\Delta E_{00} > 6.0$         | 确定存在明显的颜色突变（如红花与黄花），直接作为新类别候选。 |

### 5.2 特征提取与距离度量参数

| **参数项**           | **设置值**                | **业务含义与配置说明**                                       |
| -------------------- | ------------------------- | ------------------------------------------------------------ |
| **特征表达形式**     | 多主色向量 + 面积权重占比 | 废弃高维直方图。使用 `[(L, a, b), weight]` 形式表达。        |
| **主色提取算法**     | MiniBatchKMeans           | 兼顾计算效率与主色收敛性，适合工业级流式处理。               |
| **主色提取数量 (K)** | 3 ～ 5 个                 | 完整覆盖花卉主体渐变色及核心过渡色所需的基本数量。           |
| **高维降维算法**     | 无（废除 PCA）            | 保持 CIELAB 空间的物理语义不被破坏，保留完整的色彩可解释性。 |
| **复合距离度量**     | 陆地移动距离（EMD）       | 以 ΔE00 为 Ground Metric，定量评估两个多主色集合之间的匹配成本。 |

### 5.3 聚类与动态流转参数

| **参数项**           | **设置值**                          | **业务含义与配置说明**                                       |
| -------------------- | ----------------------------------- | ------------------------------------------------------------ |
| **核心聚类算法**     | HDBSCAN                             | 基于密度的聚类算法，支持传递预计算距离矩阵。                 |
| **距离度量模式**     | `metric='precomputed'`              | 强制接收 EMD 距离矩阵，实现离线与在线度量标准的完全闭环统一。 |
| **min_cluster_size** | 15 ～ 50                            | 形成一个稳定花卉新颜色类别的最小样本堆积量。                 |
| **min_samples**      | 5 ～ 15                             | 控制聚类簇的严苛程度，数值越大对噪声的容忍度越高、聚类越保守。 |
| **在线匹配机制**     | 最近原型匹配（Nearest Prototype）   | 依据新样本与各类别原型的最低 EMD 距离进行业务归流。          |
| **未知池触发条件**   | 样本量大于 500 条 或 占总量大于 10% | 达到该阈值时自动激活后台局部重聚类任务。                     |

## 六、方案特点

- **度量空间绝对统一**：离线聚类和在线识别全部寄宿在“ΔE00 + EMD”的非线性视觉感知距离下，彻底消除了“离线用欧氏距离、在线用色差公式”所带来的空间错配与误判隐患。
- **高解释性的特征表达**：用“主色点+权重”替代传统的抽象直方图，每一个类别原型都可以直接还原成肉眼可见的几块核心色彩组合，极大地降低了业务人员的维护与审校成本。
- **光照与阴影鲁棒性**：通过调整 ΔE00 公式中明度通道（L）的权重惩罚，使算法天然对拍摄时的天气、光照起伏及花瓣阴影具备很强的免疫力。
- **无标签漂移的动态扩展**：重聚类机制被严格限制在未知类缓冲池内部，采用局部增量挂载的方式吸收新类别，存量老类别的中心和边界不会受到干扰，保证了系统长期运行的稳定性。
